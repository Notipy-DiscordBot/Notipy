🚀 So how do we install the Discord bot?

Notipy works without a separate server—
just invite the bot and you're ready to go.

1️⃣ Invite the Discord Bot

Click "Invite Bot" on the homepage main screen!

Next

2️⃣ Set Admin Role

Assign an admin role (select NotiPy) to the invited bot to use commands.

Next

3️⃣ Register Notion Token

Generate a Notion token from your workspace and register it with the bot.

Next

4️⃣ Link Your Database

Select the Notion database to link and connect it to a Discord channel.

5️⃣ Use various commands!

Notipy offers a range of commands for different features. Try them out!

/help Shows commonly used commands.
/setup set-admin-role Sets the admin role.
/setup view Displays current configuration.
/setup webhook Sets current channel to receive webhooks.
/ticket create Creates a ticket.
/ping Checks the bot’s response speed.
