❓ NotiPy팀은 왜 디스코드를 택했나요?

Zoom, Google Meet와는 달리
디스코드는 시간 제한 없는 자유로운 회의가 가능합니다.
그리고 회의가 끝난 이후에도 공지, 대화, 자료공유까지 지속적인 협업이
이루어질 수 있는 공간이기 때문입니다.
바로 이런 이유로, 저희는 디스코드를 택했습니다.

>> 클릭해서 다음 보기 >>

🚀 그럼 이제 디스코드봇 설치 방법을 알아볼까요?

Notipy는 별도 서버 설치 없이,
초대만 하면 사용할 수 있는 봇입니다.

계속 보기

1️⃣ 디스코드 봇 초대하기

홈페이지 메인화면에서 디스코드 봇 불러오기를 클릭해주세요!

다음

2️⃣ 관리자 역할 설정

초대한 봇에게 명령어 사용을 위한 관리자 역할(NotiPy 선택)을 부여해주세요.

다음

3️⃣ Notion 토큰 등록

사용자의 워크스페이스에서 Notion 토큰을 발급하고 등록합니다.

다음

4️⃣ 데이터베이스 연동

봇과 연결할 Notion 데이터베이스를 선택하고 채널에 연동하세요.

다음

5️⃣ 그 외 다양한 명령어들도 존재해요!

NotiPy는 다양한 기능을 명령어로 제공해요. 사용해보세요!

/도움말 자주 쓰는 명령어를 보여줍니다.
/설정 관리자역할 설정 관리자 역할을 설정합니다.
/설정 보기 현재 설정을 확인합니다.
/웹훅 설정 현재 채널로 웹훅을 받도록 설정합니다.
/티켓생성 티켓을 생성합니다.
/핑 봇의 응답 속도를 확인합니다.
활용사례 살펴보기

🎯 이런 순간, Notipy를 쓰면 좋아요!

단순한 알림을 넘어, 실제 협업 상황에서 생산성을 높일 수 있도록 설계되었습니다.

다음

🚨 갑작스러운 이슈가 생겼을 때

버그나 긴급한 수정 요청을 Notion에 등록하면,
Discord 채널에 스레드가 자동 생성되어
팀원들과 빠르게 논의하고 해결할 수 있어요.
✨ GitHub 커밋 알림도 연동 가능해, 코드 변경도 실시간으로 추적돼요!

다음

📋 업무 분장이 필요할 때

새로운 프로젝트가 생기면 Notion에 등록만 하세요.
Discord에 실시간으로 전달되어 팀원 모두가
현재 해야 할 일을 한눈에 파악할 수 있어요🔎

다음

🙋‍♀️ 새 팀원이 들어왔을 때

새 팀원이 GitHub 프로필 URL을 입력하면,
Notipy가 해당 사용자의 활동 내역과 기술 스택을 LLM으로 요약해줍니다.
Discord 명령어를 통해 요약 결과를 확인할 수 있어요!
서로의 강점을 빠르게 이해하고, 프로젝트 배치를 더 효율적으로 할 수 있죠🙌🏻

처음부터 다시 보려면 상단 동그라미 클릭!
